# FrontalierFiscal 🇱🇺

Calculateur d'optimisation fiscale pour frontaliers Belgique-Luxembourg.

## À propos

FrontalierFiscal aide les salariés frontaliers à :
- Calculer leur impôt réel (double imposition BE-LU)
- Découvrir leurs leviers d'optimisation fiscale
- Voir combien ils peuvent économiser par an

Basé sur les barèmes officiels ACD Luxembourg 2025.

## Démarrage

Va sur https://frontalier-fiscal.vercel.app

## Comment ça marche

1. **Situation** : Entre ton salaire, télétravail, voiture, etc.
2. **Résultats** : L'app calcule ton impôt et ton net mensuel
3. **Optimisation** : Découvre tes leviers pour économiser

## Technologie

- React 18 (interface)
- Vercel (hébergement gratuit)
- Barèmes 2025 (ACD Luxembourg + Code belge)
